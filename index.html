WHOISBot, c'est quoi ?

WHOISBot est un bot Discord Open-Source dédié aux utilisateurs avancés. Vous avez des fonctionnalités comme le ping d'adresse IP, les informations sur une Zone DNS ou encore la décomposition d'une adresse IP.

Fonctionnalités

  • Recherche WHOIS
  • Conversion décimal / héxadécimal / octal / binaire
  • Ping d'adresse IP
  • Décomposition d'une zone DNS

Utiliser WHOISBot

WHOISBot existe en version publique, sans aucune configuration. Vous pouvez également personnaliser le code de WHOISBot pour en faire votre propre application !
